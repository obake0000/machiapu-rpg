## 📜 ストーリー

**深夜1時、勇者の自宅。**

スマートフォンが青く光る。マッチングアプリの通知。
タップした瞬間、勇者はアプリの世界に吸い込まれる。

そこにいるのは——

- 店指定で予算を聞いてくるトロール
- ホテル直行のサキュバス
- プロフ写真と別人の加工トロール
- 「上の人紹介する♡」のマルチ勧誘ちゃん
- 水星逆行で会えない占い狂いガール
- 「死にたい今すぐ電話」のメンヘラ依存
- 指輪のタンラインが見える既婚バレ女

最深部には、**「仲良くなったらね」を札のように手元に持つ死神**が待つ。

死神を退けた先に、ようやく現れる「**普通の子**」。
普通の子は言う——「あ、ここまで来てくれたんですね」。

ふたりは、定食屋で普通のごはんを食べる。
勇者は、ようやくアプリを閉じることができた。

> 盛りも、煽りも、条件交渉も、副業勧誘も、占いの話もない。
> ただ、いつものごはんの話をした。

## ⚔️ 戦闘システム

### 基本ループ
1. 雑魚エンカウント → 戦闘
2. **撃退するたびに必ずレベルアップ**（攻撃+3／最大HP+12）
3. ザコ3体倒すとボス戦の強さに到達
4. ボス（死神）撃破 → 真エンドルートへ

### 4つのコマンド
| コマンド | 内容 |
|---|---|
| 💋 **口説く** | 10%で響く、それ以外は華麗にスルーされHPダメージ。ボスのみ親密度ゲージを稼げる |
| 🍽️ **おごる** | 所持金で HP回復（最大80% + α）。敵を満足させ少しダメージも入る |
| 📸 **撮って通報** | 通常攻撃。スクショ送信でダメージ。最強の正攻法 |
| 🏃 **逃げる** | 75%で離脱（ボスからは逃げられない） |

### ボス戦：死神（永遠に近づけない）
- 親密度システム：口説く / おごるで 0→3 まで蓄積
- 親密度3未満では「撮って通報」が通らない（"仲良くなったらね"）
- HP 80・攻撃10、4種類の心が削れる攻撃

- アプリを閉じた

- ボス戦に負けた
- 翌朝、財布軽い、LINE未読、アプリだけは開いている

### 💀 GAME OVER：マチアプ疲れ
- HP 0
- 沼の連続ブロックで力尽きた

## ❓ FAQ素案

**Q: 何分でクリアできる？**
A: 最短3分、寄り道込みで7-10分。

**Q: ダウンロード必要？**
A: 不要。ブラウザ1枚（HTMLファイル）で起動します。

**Q: スマホで遊べる？**
A: 縦持ちで動きます。下部に仮想パッドを表示。

**Q: アダルト要素は？**
A: ありません。語感はブラックですが描写は健全。

**Q: マッチングアプリ批判？**
A: いいえ。アプリ自体ではなく "盛り" "条件交渉" "占い狂い" 等の **人類あるある** をネタにした笑い供養ゲームです。

**Q: 続編予定は？**
A: 「ガルバファイト！（夜のキャストとの会話バトル）」「歌舞伎町ファイター（殴らないサバイバル）」など同シリーズ展開中。
